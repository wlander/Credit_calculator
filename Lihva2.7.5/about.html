Описание программы:
Возможность расчета по следующим параметрам:
- расчет аннуитентной и дифференцированной формы кредита.
- расчет по типу: расчет платежей, расчет суммы кредита и платежей по доходу, расчет суммы кредита и платежей по ежемесячному платежу
- расчет по валюте: рубль, доллар США, евро.



Возможность построения графиков по следующим параметрам:
- по ежемесячному платежу;
- по ежемесячной выплате процентов;
- по ежемесячному погашению основного долга;
